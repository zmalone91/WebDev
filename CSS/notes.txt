Almost everything in CSS follows the basic pattern:

selector {
    property: value;
}

ex:

h1 {
    color: purple;
}

Inline CSS is not typically recommended

<link> CSS to html page in the head element

<head>
    <title>Demo HTML</title>
    <link rel="stylesheet" href="my_styles.css">
</head>
